Solar Powered IOT Based Sanitary Napkin Vending Machine:

The work aims at installing an automatic napkin dispenser on dropping of coins in toilets and places with the feature of intimating the person concerned through notification on an application for refilling the Napkins in the machine.

The whole system will be operated with the use of solar energy.

Detecting Phishing Websites Using Machine Learning:

The work aims to create a platform for the users to identify whether a website is a legitimate or not and report if it is a malicious one.

User can also add phishing website URL or fake website URL into system where system could access and scan the phishing website and by using machine learning algorithm, it will add new suspicious keywords to database.
